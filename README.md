# Our Space 💚

A private, custom Progressive Web App (PWA) hub and application ecosystem built as a dedicated personal space. The user interface features a serene emerald/sage green palette built entirely around a custom color profile (`#40826D`).

## 🗂️ Project Structure

*   **`/` (Root)**: The master hub portal (`index.html`), PWA profile (`manifest.json`), and OneSignal service worker infrastructure (`OneSignalSDKWorker.js`) coordinating application caching and installation configurations.
*   **`/4ever-timer/`**: A precision, real-time counter logging every elapsed second since the relationship milestone on **May 21st, 2026**. Features an integrated, cross-platform "I Miss You" instant push notification emitter.
*   **`/wybmgf/`**: The original interactive project mini-game files.

## 🚀 Core Features

1.  **Native Standalone Mobile App (PWA)**: Bundles the entire multi-folder web workspace into an installable mobile utility. When added to a smartphone home screen, it completely masks regular browser interfaces, address bars, and navigation frames.
2.  **Two-Way Device Signaling**: Leverages serverless API fetches directly from the client side to target alternate user registration tags. Pressing the interaction button instantly triggers a high-priority lock-screen alert on the partner's device.
3.  **Dynamic Interface Attunement**: Parses incoming URL routing parameters to seamlessly transform app greetings, button targets, and specific iOS installation prompt workflows.

## 🔗 Device Access & Query Routing

The workspace conditionally alters operational states depending on the URL query string applied:

*   **Your Personal View**: 
    `https://rhishikeshjeya-design.github.io/our-space/?user=me`
*   **Her Custom View**: 
    `https://rhishikeshjeya-design.github.io/our-space/?user=her` *(Triggers custom Safari onboarding prompts explaining how to tap 'Add to Home Screen')*

## 🛠️ Built With

*   **Hosting Engine**: GitHub Pages (SSL Enforced)
*   **Frontend Architecture**: Vanilla HTML5 / CSS3 / JavaScript
*   **Notification Gateway**: OneSignal Web Push REST API
*   **PWA Compliance Protocol**: W3C Web App Manifest
